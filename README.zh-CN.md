# Liquid Glass MP3 可视化播放器

一个纯前端 MP3 可视化播放器：中心旋转圆形图标 + 环形炫彩频谱线 + 苹果风格 Liquid Glass 控制面板。

English version: [README.md](README.md)

## 功能特性

- 屏幕中心圆形 MP3 图标，360° 持续旋转
- 图标外圈炫彩环形音乐频谱线
- 苹果 Liquid Glass 风格控制区
- 选择本地 MP3 并循环播放
- 选择自定义背景图
- 选择自定义音乐图标（唱片封面）
- 图标透明度滑杆
- 先“配置完成”再允许播放
- 提供“开始 / 暂停”控制
- 支持“隐藏 / 显示”控制面板
- 内置中英文界面切换

## 快速开始

1. 克隆仓库到本地。
2. 使用浏览器打开 `html/HTML-MusicPlayer-Zh-cn.html`。
3. 建议浏览器：Chrome / Edge 最新版。

无需安装依赖，无需构建。

## 使用流程

1. 点击 `选择 MP3`，选择本地 `.mp3` 文件。
2. 可选配置：
- `选择背景图`
- `选择音乐图标`
- 调整 `图标透明度`
3. 点击 `配置完成`。
4. 点击 `开始` 播放，点击 `暂停` 暂停。
5. 可用 `隐藏控制面板` / `显示控制面板` 切换面板显示。
6. 点击 `Language` 按钮可切换中文/英文界面。

## 说明

- 浏览器自动播放策略可能要求先进行一次用户交互。
- 本地文件通过 `ObjectURL` 加载，仅在本地浏览器会话中使用。
- 频谱线持续动画，播放音乐时会按频谱实时变化。
